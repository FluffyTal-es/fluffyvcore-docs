🧰 A Starting Kit

Everything in the Athena Framework is built to help you script a game mode faster. Using Athena requires some programming knowledge, and is not a plugin and play game mode.

🛠️ Powered by TypeScript

Using modern libraries like TypeScript, Vite, SWC, Vue 3, and MongoDB. A nearly seamless development environment for faster results.
