# FluffyVCore API

The API has many different options but pay attention to the context at the top to see where the API pathways are available.

## Server Import

```ts
import * as FluffyVCore from '@FluffyVCore/server';
```

## Client Import

```ts
import * as FluffyVCore from '@FluffyVCore/client';
```

## Shared Import

Use any of the folders in the shared directory to import it anywhere.

```ts
import * as Foo from '@FluffyVCore/shared/foo';
```
